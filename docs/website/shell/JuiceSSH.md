## 使用

管理连接 - 认证 - 新建认证

填好后，右上对勾 `√` 确定

::: tip 说明
用户名：root

密码：(你设置的服务器密码)
:::

![](/shell/JuiceSSH/JuiceSSH-03.png)



返回 `连接` 页面右下 + 号，新建连接

填好后，右上对勾 `√` 确定

::: tip 说明
地址：(你的IP地址)

认证：(选择刚才的认证)
:::

![](/shell/JuiceSSH/JuiceSSH-04.png)
