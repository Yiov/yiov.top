# 血常规化验单解读




## 简介

由于血常规检查方便易行、所含的信息量巨大，所以是临床医生最常用的辅助检查手段之一

血常规检查包括有红细胞计数（RBC）、血红蛋白（Hb）、白细胞（WBC）、白细胞分类计数及血小板（PLT）等，有助于诊断感染、贫血、白血病、血小板减少、失血等多种疾病

三大系统：

* 红细胞
* 白细胞：中性粒细胞、嗜酸性粒细胞、嗜碱性粒细胞、淋巴细胞、单核细胞
* 血小板
