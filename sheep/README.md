## 素材清单


用MT管理器获取了本地素材路径，没想到竟然是整合在一张图上的

于是用PS拼接，`收纳了所有表情包，包含下架的坤坤和悟空等`，以及过场动画

算是比较全的了

![](https://ghproxy.com/https://raw.githubusercontent.com/Yiov/notes/main/sheep/exeample.png)

</br>

* gif格式表情包 — 52张

* title图 — 6张

* wallpaper — 7张

* 原素材 — 10张

* 说明文档 — 1份

* png格式表情包 — 52张

</br>
