# 静态托管介绍

> 更新：2024-7-6


## 简介

静态托管就是将静态网站的内容（如HTML、CSS、JavaScript等文件）托管在它的服务器上

::: details 与动态有什么区别
动态和静态的最主要的区别就是数据交互

* 动态：有数据库，且数据可以随时增减，一般网站是可以注册账户的

* 静态：无数据库，不可注册用户
:::
