#  iPhone越狱工具介绍




## 简介

越狱的目的简单来讲，就是通过安装各种插件来追求更个性化、更安卓化

随着苹果漏洞的完善，越狱在不久的将来，可能会不复存在

## 越狱工具

带 ⭐ 的是有教程的，点击即可跳转

::: details 关于有根和无根
有根是 `读写` 系统权限，无根是`只读` 系统权限

主要原因是由于苹果漏洞的修复，越狱已经举步维艰了
:::
